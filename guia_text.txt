En esta actividad de aprendizaje usted Generará script DDL (lenguaje de definición de datos) de  la 
base de datos en SQL .  
Almacenar información es uno de los usos más frecuentes que una organización o empresa realiza 
día a día en un computador. La construcción e implementación de bases de datos permite que la 
organización pueda almacenar la información relevante del negocio a  través de la construcción de 
una base de datos que se realiza por medio del lenguaje de definición de datos (DDL = Data 
Definition Language).

El lenguaje de definición de datos permite establecer y/o modificar el esquema relacional, es decir, 
añadir, borrar o actualizar atributos, tablas, índices, etc .

CREATE TABLE  
Permite crear una tabla con campos con los tipos de dato, solo funciona si ya creo la base de datos 
y la tiene en uso, ejemplo 1:  
Producto  
codigoProducto  nombreProducto  precioProducto

CREATE TABLE Producto(  
      codigoProducto  SMALLINT NOT NULL PRIMARY KEY , 
      nombreProducto VARCHAR(40) NOT NULL , 
      precioProducto  MEDIUMINT NOT NULL  
); 
 
Explicación:  
NOT NULL = Significa que obligatoriamente debe llevar algún  dato al momento de hacer INSERT  
SMALLINT, MEDIUMINT, VARCHAR(40) = Son tipos de datos.  
PRIMARY KEY  = Es la llave primaria cuyo valor al hacer INSERT no se debe repetir.

5. GLOSARIO DE TÉRMINOS  
DDL: Data Definition Language  o Lenguaje de definición de datos entre los que podemos encontrar 
los comandos CREATE, ALTER, DROP  
DML : Data Manipulation Language o Lenguaje de manipulación de datos entre los que podemos 
encontrar INSERT INTO, UPDATE SET, DELETE FROM, SELECT FROM  
Script: Secuencia de comandos que permite ejercutar una serie de procesos, en este caso una 
sentencia completa para crear, alterar, insertar, actualizar, eliminar, seleccionar, entre otros se le 
denomina script.
